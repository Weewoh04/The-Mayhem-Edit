# Image Guide

Recommended dimensions:

- Collection card: 1200 x 900 pixels
- Collection hero: 1600 x 1000 pixels
- Pinterest pin: 1000 x 1500 pixels
- Product image: 1000 x 1000 pixels with a clean background

Replace these starter SVG placeholders with approved images you own, generated images, or product images you have permission to use. Do not scrape or hotlink Amazon product photos.

Suggested folders:

- `public/images/collections/` for collection cards and hero graphics
- `public/images/products/` for square product images
- `public/images/pinterest/` for tall Pinterest pin graphics
